### How to configure brave

Launch brave

Go in `Settings`
![Brave settings](doc/img/brave_settings.png)

Go in `Sync`
![Brave sync](doc/img/brave_sync.png)

Click on `Manage your synced devices`
![Brave Manage synced devices](doc/img/brave_manage-sync.png)

Use sync code (send to Mee6 by discord for me)

Check all options except `Open tabs`
![Brave options check](doc/img/brave_checks.png)
