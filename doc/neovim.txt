Make sure you have installed vimplug

Start by creating the neovim config directory: `mkdir ~/.config/nvim/`
Then go in it: `cd ~/.config/nvim`

Copying the content of `Dotfiles/nvim`(on gihub, clone repository) in it
Setup onedark colorscheme: `git clone https://github.com/joshdick/onedark.vim.git /tmp/onedark.vim; cp /tmp/onedark.vim/colors -r ~/.config/nvim/colors; cp /tmp/onedark.vim/autoload -r ~/.config/nvim/autoload`
Setup correct font to get icons: `git clone https://github.com/ryanoasis/nerd-fonts.git /tmp/nerd-fonts; cd /tmp/nerd-fonts; ./install.sh Hack`

Launch neovim: `nvim`
Type: `:PlugInstall`
