Create new ssh key and add it to github and gitlab:

Generate new ssh key: `ssh-keygen -t ed25519 -C "chervaud@hp_manjaro"`
Add ssh key to ssh-agent: `ssh-add ~/.ssh/id_ed25519`

Go to `https://github.com/settings/keys`
Click `New SSH key`
Set a title, example: 'HP Laptop Manjaro'
Copy your ssh public key (location: `~/.ssh/id_ed25519.pub`)
Cleck `Add SSH Key`
Click `Enable SSO`

Go to `https://gitlab.com/-/profile/keys`
Copy your ssh public key (location: `~/.ssh/id_ed25519.pub`)
Set a title, example: 'HP Laptop Manjaro`
Add an expiration date if you want
Click `Add Key`
