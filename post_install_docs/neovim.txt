Make sure you have installed vimplug

Start by creating the neovim config directory: `mkdir ~/.config/nvim/`
Then go in it: `cd ~/.config/nvim`

Copying the content of `Dotfiles/nvim`(on gihub, clone repository) in it
Setup onedark colorscheme: `git clone https://github.com/joshdick/onedark.vim.git /tmp; cp /tmp/onedark.vim/colors -r ~/.config/nvim/colors; cp /tmp/onedark.vim/autoload -r ~/.config/nvim/autoload`

Launch neovim: `nvim`
Type: `:PlugInstall`
